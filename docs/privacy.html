2. 채팅 내용의 처리 (미보관 원칙)

채팅 메시지(텍스트·그림)는 다음과 같이 처리됩니다.

  • 메시지는 수신자에게 전달하기 위한 목적으로만 서버를 일시 경유하며, 수신 확인 즉시 서버에서 삭제됩니다.
  • 수신자가 접속하지 않아 전달되지 못한 메시지도 최대 24시간 후 서버에서 자동 삭제됩니다.
  • 전달된 메시지는 각 이용자의 기기 내부에만 저장되며, 기기에서도 24시간 후 자동 삭제됩니다.
  • 따라서 서비스 운영자는 지나간 채팅 내용을 열람하거나 복구할 수 없습니다.

3. 연락처 정보의 처리

  • 연락처 접근 권한은 이용자가 친구 찾기 기능을 사용할 때만 요청됩니다.
  • 연락처의 전화번호는 기기 안에서 일방향 해시(SHA-256)로 변환된 뒤 서버와 대조되며, 전화번호 원본과 이름은 서버로 전송되지 않습니다.
  • 친구 목록은 서버에 저장되지 않고 이용자 기기에만 저장됩니다.
  • 전체 자동 등록과 직접 선택 등록 중 이용자가 방식을 선택할 수 있습니다.
